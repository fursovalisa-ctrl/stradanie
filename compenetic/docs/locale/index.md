# Локализация

::: tip Что это?
Система локализации в SkillGrid построена на базе i18next и обеспечивает типобезопасный перевод компонентов на разные языки.
:::

## Особенности

- Типобезопасность через TypeScript
- Модульная структура переводов
- Автоматическая генерация типов
- Интеграция с компонентами
- Поддержка множества языков

## Использование

```tsx
import { useTranslation } from 'react-i18next';

function MyComponent() {
  const { t } = useTranslation();
  
  return (
    <div>
      {t('myComponent.title')}
    </div>
  );
}
```

## Структура

Каждый компонент может иметь свой файл локализации:

```
ComponentName/
└── ComponentName.i18n.ts  # Файл с переводами
```
